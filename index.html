Selected Projects

Autonomous Unmanned ATV – ROS · Pixhawk · PLC Integration

ROS Pixhawk 6C LiDAR PLC Autonomous Driving

Developed a full autonomous vehicle platform by integrating ROS, Pixhawk 6C, a Quanergy M8 3D LiDAR, and industrial PLC motor control into a unified real-time navigation system.

  • Processed dense pointcloud2 data using PCL to detect obstacles and evaluate safe gaps.
  • Implemented ROS decision nodes that generate steering and speed commands based on LiDAR geometry.
  • Used MAVROS to send RC Override PWM commands to Pixhawk, enabling real-time control while preserving AUTO missions.
  • Designed a hardware chain (Pixhawk PWM → Arduino measurement → DAC voltage → PLC) to drive industrial AC servo motors.
  • Verified algorithms in Gazebo (r1_rover) before deploying to the real ATV platform.

Autonomous Baseball Pitching Robot for 3×3 Target Grid

Computer Vision Robotics YOLOv8 BLDC Launcher

Designed and implemented an autonomous baseball launcher capable of hitting a 3×3 target grid from 5 m using real-time target detection, physics-based trajectory modeling, and BLDC motor control. This system served as the pitching platform for the TEL Robot Combat competition.

  • Ran YOLOv8 on Raspberry Pi / Jetson Orin Nano to localize grid cells on the target board.
  • Built a projectile model including launch velocity, drag, and spin effects to compute wheel speed and launch angle.
  • Controlled dual high-speed BLDC wheels (~3000 RPM) with feedback for consistent exit velocity.
  • Computed angle offsets and trigger timing in real time to achieve accurate shots into specified grid cells.

Robot Combat System – TEL Robot Combat 2025 (Kiwi Drive Platform)

Embedded Control Kiwi Drive Robotics

Built a holonomic-drive combat robot with a Kiwi (tri-omni) drivetrain, integrating perception, motion control, and competition strategy into a unified system.

  • Implemented Kiwi-drive kinematics for omnidirectional movement and rapid reorientation.
  • Integrated sensors and control loops for precise position control under dynamic competition conditions.
  • Collaborated in a multidisciplinary team and took responsibility for motion control and systems integration.
